This plugin provides WordPress users access to over 500 offers available through blvd-media.com via the accessTool.

== Description ==

This plugin allows Blvd Media Group, LLC affiliates to use te accessTool on their WordPress sites easily.
The plugin has two features that allow a user to have it load on every page (or) on selected posts when the [blvdmedia tool=at] tag is placed. The plugin also removes the tag from your posts if the plugin is currently disabled via the plugin option window, or is set to load on every page.

== Installation ==

1. Extract the `blvd_at` folder from blvd_at.zip
2. Upload the folder `blvd_at` to the `/wp-content/plugins/` directory.
3. Activate the plugin through the 'Plugins' menu in WordPress.
4. Open the Blvd Media accessTool page, and select your preferences.

== Frequently Asked Questions ==

* How would I have it run on selected pages? *

The plugin searches for the [blvdmedia tool=at] tag and replaces it with the accessTool if this option is selected on the options page.

* How would I load the accessTool on every page? *

That option is available on the accessTool plugin options page.
